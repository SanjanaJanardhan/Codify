# Codify

A frontend prototype for a coding education platform. Includes a landing page with login and signup flows, and a home dashboard — built with vanilla HTML, CSS, and JavaScript.

The UI has a few programmer-flavoured touches: input labels like `char [] Email-ID` and `loop(Password)` to match the coding theme.

---

## Pages

- **Landing page** — login and signup forms with smooth toggle between the two views
- **Home page** — dashboard landing after auth

---

## Tech Stack

- HTML · CSS · Vanilla JavaScript
- No frameworks, no build step

---

## Getting Started

No setup needed. Just open in your browser:

```bash
git clone https://github.com/SanjanaJanardhan/Codify.git
cd Codify
open landingpage/landingpage.html
```
